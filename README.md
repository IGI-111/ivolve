Ivolve
======

Ivolve is a genetic image vectorizer which uses [simulated annealing](http://en.wikipedia.org/wiki/Simulated_annealing).

To build do:
```
  git clone https://github.com/IGI-111/ivolve.githttps://github.com/IGI-111/ivolve.git
  cd ivolve
  mkdir build
  cd build
  cmake .. && make
```
